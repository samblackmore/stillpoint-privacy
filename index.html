Privacy Policy — Stillpoint

Last updated: November 24, 2025

Stillpoint is a simple offline game built with Unity. This privacy policy explains how the game handles user data.

1. No data collected

The game does not collect, store, or transmit any personal information, either automatically or manually. Specifically, the game does not collect:

  • Names, email addresses, or account information
  • Device identifiers
  • Location data
  • Gameplay analytics or usage statistics
  • Advertising identifiers
  • Payment or financial information

The game contains no ads, no analytics SDKs, and no third-party tracking tools.

2. Offline game

The game works entirely offline. It does not require an internet connection and does not communicate with any servers.

3. No sharing of information

Because the game does not collect any information, it also does not share any information with third parties.

4. Children’s privacy

The game is suitable for all ages, including children, but it is not specifically directed at children. Since no data is collected from any users, the game is designed to comply with applicable children’s privacy protections (for example, COPPA in the United States) by design.

5. Changes to this policy

If the game is updated to include features that involve data collection, analytics, or online communication, this privacy policy will be updated accordingly. An updated version of this document will be posted here.
